# 🎓 Student Management System

A simple yet effective console-based Java application for managing student records. This system allows you to perform CRUD operations (Create, Read, Delete, Update) on student data with built-in input validation.

## ✨ Features

- **Add Students** - Add single or multiple students at once
- **Delete Students** - Remove students by their unique ID
- **Replace Students** - Replace existing student information
- **View All Students** - Display complete list of enrolled students
- **Input Validation** - Prevents invalid data entry (names with digits, negative ages, etc.)
- **Error Handling** - Graceful handling of invalid inputs (non-numeric values, negative numbers)

## 🚀 Getting Started

### Prerequisites

- Java JDK 17 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line
